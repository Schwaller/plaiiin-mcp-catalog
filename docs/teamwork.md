#### Tool: **`twchat-get_conversation`**
Get a single Teamwork Chat conversation by ID.
Parameters|Type|Description
-|-|-
`conversation_id`|`integer`|The ID of the conversation to retrieve.

*This tool is read-only. It does not modify its environment.*

---
#### Tool: **`twchat-get_current_user`**
Get the current authenticated Teamwork Chat user, including identity, counts (unread conversations/messages, mentions), and settings.
#### Tool: **`twchat-get_or_create_dm`**
Get the 1:1 direct-message conversation with a person, creating it if it does not exist yet. Returns the conversation (use its id with send_message). Use list_people to find user_id.
Parameters|Type|Description
-|-|-
`user_id`|`integer`|The ID of the person to get the direct-message conversation with.

*This tool is read-only. It does not modify its environment.*

---
#### Tool: **`twchat-list_conversations`**
List Teamwork Chat conversations the current user is a member of.
Parameters|Type|Description
-|-|-
`include_message_data`|`string` *optional*|Include the latest message in each conversation.
`page_limit`|`string` *optional*|Number of conversations to return (max 10).
`page_offset`|`string` *optional*|The index position to start retrieving results from (not a page number).
`search_term`|`string` *optional*|A search term to filter conversations by title.
`sort`|`string` *optional*|Sort order for the returned conversations.
`status`|`string` *optional*|Filter by conversation status.
`type`|`string` *optional*|Filter by conversation type: "rooms" for group/channel conversations, "pair" for 1:1 direct messages.

*This tool is read-only. It does not modify its environment.*

---
#### Tool: **`twchat-list_messages`**
List messages within a Teamwork Chat conversation. Requires conversation_id.
Parameters|Type|Description
-|-|-
`conversation_id`|`integer`|The ID of the conversation to read messages from.
`after_message_id`|`string` *optional*|Return messages newer than this message ID (cursor).
`before_message_id`|`string` *optional*|Return messages older than this message ID (cursor).
`created_after`|`string` *optional*|Return messages created after this time.
`created_before`|`string` *optional*|Return messages created before this time.
`page`|`string` *optional*|Page number for pagination of results.
`page_size`|`string` *optional*|Number of messages per page (1-200).
`search_term`|`string` *optional*|A search term to filter messages by text content.

*This tool is read-only. It does not modify its environment.*
